# MovieLibrary – Base de proyecto docente

Este proyecto contiene una **estructura base** para el Taller Práctico 4 – MovieLibrary:

- Configuración Gradle del módulo `app`.
- Activity principal con `NavHostFragment`.
- Tres fragments vacíos (`MovieListFragment`, `MovieDetailFragment`, `MovieEditFragment`) con layouts simples.
- Clase de dominio `Movie`.
- Navigation Graph (`nav_graph.xml`) con las pantallas registradas.

La lógica de negocio, Room, Repository, ViewModel, observación de LiveData y navegación con argumentos queda a cargo de los estudiantes, siguiendo el enunciado del taller.
